Shortcuts

Aspia supports hot keys that allow you to call some of the features of the program without using the mouse.

The following shortcuts:

  • Ctrl + Q - Exit from program.
  • Ctrl + S - Report Saving.
  • Ctrl + A - Select all rows in information window.
  • Ctrl + C - Copy selected row(s).
  • F5 - Updating the current open page.
  • F1 - Displays help on using the program.
