Using the command line

Aspia can run from the command line. This feature allows you to save a report in the selected format and in a user-specified directory without launching the graphical interface.

aspia.exe [file_path].[file_ext] [-nonav] [-help]

file_path - path and file name. For example, C:\Temp\report.htm
file_ext - file extension. You must correctly specify the extension, since it is determined by the file format, which will be save the report. The following extensions (formats):

  • htm - HTML-File
  • txt - Plain Text File
  • csv - Table with ";" separator
  • rtf - Rich Text Format
  • ini - INI-File
-nonav - Option to turn off add to the file contents of the report.
-help - Displays help on using Aspia.

Examples of use:
  • Displays help:
    aspia.exe -help
  • Save the report without the contents to the directory C:\Temp with the name of the test file in the format of HTML:
    aspia.exe C:\Temp\test.htm -nonav
  • Save the report table of contents in the directory C:\Temp with the name test2 file format RTF:
    aspia.exe C:\Temp\test2.rtf
