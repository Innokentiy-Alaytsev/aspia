Saving a Report

In many cases, you want to save the data for later viewing on another computer or to compare changes in different time intervals. For these purposes Aspia has the means to save the collected data in several common file formats. Also, there are several ways to save the report.
In the main window toolbar has a button «Save Report», when clicked, displays a menu with options for conservation (Figure 1).

Figure 1

The user can choose to save:
  • Current Page
  • All Categories
  • Selected Categories

When you select a menu item «Current Page» or «All Categories» is displayed to the user a standard dialog box to save the file where it can select the desired file format, file name and directory to which the report will be saved.

After clicking the «Save» will be launched the process of saving the report, the state is displayed in the dialog box (Figure 2).

Figure 2

Selective preservation of the report shows a list of categories that will be saved to a file. Dialogue sample preservation (Figure 3) allows the user to specify the format, name and file path, as well as determine the need to add table of contents in the report.

Figure 3


When you click «Save» to save the report started with the mapping of the current state (Figure 2).
