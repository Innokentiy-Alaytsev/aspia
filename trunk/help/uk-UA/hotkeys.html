Гарячі клавіші

Aspia підтримує гарячі клавіші, які дозволяють викликати деякі функції програми без використання миші.

Підтримуються наступні гарячі клавіші:

  • Ctrl + Q - Вихід з програми.
  • Ctrl + S - Збереження звіту.
  • Ctrl + A - Виділити всі записи в вікні інформації.
  • Ctrl + C - Копіювання виділеного запису (чи записів).
  • F5 - Оновлення поточної відкритої сторінки.
  • F1 - Виклик довідки з використання програми.
