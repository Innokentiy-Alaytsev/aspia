FAQ
  1. I found a bug in the program. How do I report it?
    Write about it to support@aspia.ru detailing the most errors and the actions that preceded it. This will help to make Aspia better.
  2. How to support the project?
    Aspia - open source software with source code that is available to everyone. You can do your testing, bug fixes, translation of software into other languages??, writing documentation or implement new features.
  3. Is it possible to support the project financially?
    Aspia - non-profit project, which does not aim to profit, but to maintain its infrastructure funds are required. You can support the project by moving some money into our bank account.
  4. What language is written in the program?
    Aspia completely written in C language using WinAPI and NativeAPI.
  5. Already have similar programs. Why another one?
    Aspia - one of a kind that is completely free and open. You can not be afraid of the hidden features you can add your own desired functionality or fix bugs.
  6. Where to find the source code?
    All the source code of the program are available on the website. There is also a svn-repository Aspia on Google Code.
  7. What is the difference Portable-version of the standard? Portable-version does not require installation and can work with an external hard drive or flash card. The configuration file is located at the same time in the same folder with the program, but not in the user profile.
  8. How to «transform» the installed version to Portable? To do this, copy the program folder (usually «C:\Program Files\Aspia») at the right place for you and create a directory with the program a blank file with name «portable» (without extension). The configuration program will continue in the same directory as the program itself, and the configuration file, which is located in the user profile will be ignored.
  9. Where Aspia stores the configuration file?
    The configuration file Portable-version is in the same folder with the program. For the regular version, it is stored in the user's home directory.
    In Windows XP/2003: C:\Documents and Settings\user_name\Application Data\aspia.ini
    In Windows Vista/2008/7/2008 R2: C:\Users\user_name\AppData\Roaming\aspia.ini
  10. Did not find an answer to your question?
    Write it to support@aspia.ru.
