Aspia - программа для сбора и анализа информации о компьютере и установленном программном обеспечении для ОС Windows.

Горячие клавиши
Выход из программы:                   Ctrl + Q
Сохранить отчет:                      Ctrl + S
Копировать выделенную строку:         Ctrl + C
Обновить текущую страницу:            F5
Вызов диалогового окна "О программе": F1


Работа из коммандной строки
aspia.exe [file_path] [-nonav] [-help]
file_path - указывает путь по которому нужно сохранить файл с отчетом
-nonav - при указании этого параметра меню для навигации в файл отчета не добавляется
-help - вывод справки по использованию программы из командной строки
